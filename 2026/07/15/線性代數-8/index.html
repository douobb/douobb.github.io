Householder 轉換

定義

設 u⃗∈Rn×1\vec{u}\in\mathbb{R}^{n\times1}u∈Rn×1，且 ∥u⃗∥=1\lVert\vec{u}\rVert=1∥u∥=1。定義 H=I−2u⃗u⃗TH=I-2\vec{u}\vec{u}^TH=I−2uuT
稱 HHH 為一個 Householder matrix

Note

設 H=I−2u⃗u⃗TH=I-2\vec{u}\vec{u}^TH=I−2uuT，∥u⃗∥=1\lVert\vec{u}\rVert=1∥u∥=1，並令 U=span⁡{u⃗}U=\operatorname{span}\{\vec{u}\}U=span{u}。則 Λ(H)={−1,1},det⁡(H)=−1\Lambda(H)=\{-1,1\}, \det(H)=-1Λ(H)={−1,1},det(H)=−1
若 PH(t)P_H(t)PH​(t) split over FFF，則 PH(t)=(t+1)(t−1)n−1P_H(t)=(t+1)(t-1)^{n-1}PH​(t)=(t+1)(t−1)n−1

  • Hu⃗=−u⃗H\vec{u}=-\vec{u}Hu=−u，故 E−1=U=span⁡{u⃗}E_{-1}=U=\operatorname{span}\{\vec{u}\}E−1​=U=span{u}
  • 若 x⃗∈U⊥\vec{x}\in U^\perpx∈U⊥，則 Hx⃗=x⃗H\vec{x}=\vec{x}Hx=x，故 E1=U⊥E_1=U^\perpE1​=U⊥
  • 若 {x⃗1,…,x⃗n−1}\{\vec{x}_1,\ldots,\vec{x}_{n-1}\}{x1​,…,xn−1​} 為 U⊥U^\perpU⊥ 之一 basis，則 {u⃗,x⃗1,…,x⃗n−1}\{\vec{u},\vec{x}_1,\ldots,\vec{x}_{n-1}\}{u,x1​,…,xn−1​} 為 HHH 之 nnn 個 LI eigenvectors，因此 HHH 可對角化

Householder Like

定理

  • 設 A,B∈Fn×nA,B\in F^{n\times n}A,B∈Fn×n，且 PAB(t)P_{AB}(t)PAB​(t)、PBA(t)P_{BA}(t)PBA​(t) 皆 split over FFF
    則 λ(AB)=λ(BA)\lambda(AB)=\lambda(BA)λ(AB)=λ(BA)
    即 PAB(t)=PBA(t)P_{AB}(t)=P_{BA}(t)PAB​(t)=PBA​(t)

  • 設 A∈Fm×nA\in F^{m\times n}A∈Fm×n，B∈Fn×mB\in F^{n\times m}B∈Fn×m，且 m≥nm\geq nm≥n

  • x⃗\vec{x}x 為 AAA 之 eigenvector，對應 eigenvalue 為 1+y⃗Tx⃗1+\vec{y}^T\vec{x}1+y​Tx

奇異值分解

定義

設 A∈Fm×nA\in F^{m\times n}A∈Fm×n。若 A=UΣVHA=U\Sigma V^HA=UΣVH
其中 U∈Fm×mU\in F^{m\times m}U∈Fm×m、V∈Fn×nV\in F^{n\times n}V∈Fn×n 為 unitary matrices，則稱上式為 AAA 之 singular value decomposition (SVD)

Σ∈Fm×n\Sigma\in F^{m\times n}Σ∈Fm×n 為 rectangular diagonal matrix。
令 r=rank⁡(A)r=\operatorname{rank}(A)r=rank(A)、p=min⁡{m,n}p=\min\{m,n\}p=min{m,n}，則 σ1≥σ2≥⋯≥σr>0,σr+1=⋯=σp=0\sigma_1\geq\sigma_2\geq\cdots\geq\sigma_r>0, \sigma_{r+1}=\cdots=\sigma_p=0σ1​≥σ2​≥⋯≥σr​>0,σr+1​=⋯=σp​=0
其中 σ1,…,σp\sigma_1,\ldots,\sigma_pσ1​,…,σp​ 稱為 AAA 之 singular values

定理

設 A∈Fm×nA\in F^{m\times n}A∈Fm×n，則 AAA 可作 SVD

  • 如果為扁矩陣，則求 ATA^TAT 之 SVD，之後再轉置
範例
定義

設 A=UΣVH∈Fm×nA=U\Sigma V^H\in F^{m\times n}A=UΣVH∈Fm×n 為 AAA 之 SVD。
定義 Σ+∈Fn×m\Sigma^+\in F^{n\times m}Σ+∈Fn×m 為將 Σ\SigmaΣ 所有非零 singular values 取倒數後，再轉置所得之 rectangular diagonal matrix:

Σ+=diag⁡(1σ1,…,1σr,0,…,0)\Sigma^+=\operatorname{diag}\left(\frac{1}{\sigma_1},\ldots,\frac{1}{\sigma_r},0,\ldots,0\right) Σ+=diag(σ1​1​,…,σr​1​,0,…,0)

定義 A+=VΣ+UHA^+=V\Sigma^+U^HA+=VΣ+UH
稱 A+A^+A+ 為 AAA 之 pseudoinverse 或 generalized inverse

Note
  • 若 AAA 可逆，則 A+=A−1A^+=A^{-1}A+=A−1
  • (A+)+=A(A^+)^+=A(A+)+=A
  • (AH)+=(A+)H(A^H)^+=(A^+)^H(AH)+=(A+)H
  • 若 AAA 為 full column rank、BBB 為 full row rank
    則 (AB)+=B+A+(AB)^+=B^+A^+(AB)+=B+A+
定理

設 A∈Fm×nA\in F^{m\times n}A∈Fm×n，x⃗∈Fn×1\vec{x}\in F^{n\times1}x∈Fn×1，b⃗∈Fm×1\vec{b}\in F^{m\times1}b∈Fm×1。
則 x⃗0=A+b⃗\vec{x}_0=A^+\vec{b}x0​=A+b
為使 ∥Ax⃗−b⃗∥\lVert A\vec{x}-\vec{b}\rVert∥Ax−b∥ 最小之 least squares solution，且在所有 least squares solutions 中，x⃗0\vec{x}_0x0​ 之 222-norm 最小

  • 若 AAA 的 columns 為 LI，則 A+=(AHA)−1AHA^+=(A^HA)^{-1}A^HA+=(AHA)−1AH
