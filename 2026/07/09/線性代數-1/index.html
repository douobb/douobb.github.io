Ch1 矩陣與線性方程組

矩陣

定義

ATA^TAT: AAA 之 Transpose
AH=A‾TA^H = \overline{A}^TAH=AT: AAA 之 Conjugate transpose

  • Symmetric matrix: AT=AA^T=AAT=A
  • Skew-symmetric: AT=−AA^T=-AAT=−A
  • Hermitian: AH=AA^H=AAH=A
  • Skew-hermitian: AH=−AA^H=-AAH=−A
性質
  • (AT)K=(AK)T(A^T)^K = (A^K)^T(AT)K=(AK)T
  • (AH)K=(AK)H(A^H)^K = (A^K)^H(AH)K=(AK)H
定義

tr(A)tr(A)tr(A): A 之 Trace (對角項總和)

性質
  • tr(AT)=tr(A)tr(A^T) = tr(A)tr(AT)=tr(A)
  • tr(AH)=tr(A‾)tr(A^H) = tr(\overline{A})tr(AH)=tr(A)
定理
  • A:m∗n, B:n∗mA: m*n,~B:n*mA:m∗n, B:n∗m, 則 tr(AB)=tr(BA)tr(AB)=tr(BA)tr(AB)=tr(BA)

反矩陣

定義
  • Left inverse 定義: 若 BA=IBA=IBA=I，則 BBB 為 AAA 的左反矩陣
  • Right inverse 定義: 若 AC=IAC=IAC=I，則 CCC 為 AAA 的右反矩陣
定理

若 AAA 具左反 BBB、右反 CCC，則 B=CB=CB=C

定義

A:n∗nA: n*nA:n∗n, 若 ∃B:n∗n s.t. AB=I=BA\exists B: n*n \text{ s.t. } AB=I=BA∃B:n∗n s.t. AB=I=BA，稱 BBB 為 AAA 之 Inverse matrix (反矩陣)，且稱 AAA 為 invertible/nonsingular

定理

A:n∗nA: n*nA:n∗n 可逆，則 AAA 之 inverse 唯一，記作 A−1A^{-1}A−1

性質

A:n∗nA: n*nA:n∗n 可逆，則

  • ATA^TAT 可逆，且 (AT)−1=(A−1)T(A^T)^{-1} = (A^{-1})^T(AT)−1=(A−1)T
  • AHA^HAH 可逆，且 (AH)−1=(A−1)H(A^H)^{-1} = (A^{-1})^H(AH)−1=(A−1)H
  • (AK)−1=(A−1)K(A^K)^{-1} = (A^{-1})^K(AK)−1=(A−1)K

基本列運算

定義

A:m∗nA: m*nA:m∗n, 基本列運算:

  • rij(A)r_{ij}(A)rij​(A) 表示交換 AAA 的第 iii 列與第 jjj 列
  • ri(k)(A)r_i^{(k)}(A)ri(k)​(A) 表示將 AAA 的第 iii 列乘以 kkk，其中 k≠0k\ne0k​=0
  • rij(k)(A)r_{ij}^{(k)}(A)rij(k)​(A) 表示將 AAA 的第 iii 列之 kkk 倍加至第 jjj 列
性質

列運算的反函數仍為列運算

  • (rij)−1=rij(r_{ij})^{-1}=r_{ij}(rij​)−1=rij​
  • (ri(k))−1=ri(1/k), k≠0(r_i^{(k)})^{-1}=r_i^{(1/k)},~k\ne0(ri(k)​)−1=ri(1/k)​, k​=0
  • (rij(k))−1=rij(−k)(r_{ij}^{(k)})^{-1}=r_{ij}^{(-k)}(rij(k)​)−1=rij(−k)​
定義

Row elementary matrix (列基本矩陣): E=r(I)E = r(I)E=r(I)

  • EEE 可逆，且 E−1=r−1(I)E^{-1} = r^{-1}(I)E−1=r−1(I)
定理
  • A→rBA \xrightarrow{r} BAr​B (列運算 rrr 將 AAA 變為 BBB)，且 E=r(I)E=r(I)E=r(I)，則 B=EAB=EAB=EA
  • A∼rBA \overset{r}{\sim} BA∼rB (AAA 列等價於 BBB)，則 ∃P\exists P∃P 為可逆矩陣，使得 PA=BPA=BPA=B

線性方程組

定義

  • Row echelon form: 一個 matrix 滿足以下條件

    • 所有零列皆在所有非零列下方
    • 每一列最左邊的非零項，在愈下方列的位置愈靠右
    • 每一列最左邊的非零項為 111

  • Reduced row echelon form: 一個 Row echelon form 滿足以下條件

    • 每一列最左邊的非零項 (首項為 111)，其所在之行的其他元素皆為 000
Note
  • 在以上型式中，非零列的最左邊非零項稱為 pivot，其所在的行稱為 pivot column
  • 每一個 matrix 皆列等價於某個 ref
  • 每一個 matrix 皆列等價於唯一 rref
  • iii pivot column 必為單位矩陣的 iii 行
定理
  • [A∣b⃗]→r[U∣c⃗][A|\vec{b}] \xrightarrow{r} [U|\vec{c}][A∣b]r​[U∣c]，則 Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b 與 Ux⃗=c⃗U\vec{x}=\vec{c}Ux=c 具有相同解集合
  • 高斯消去法即為此定理的應用: 對增廣矩陣做列運算，將其化為 ref 或 rref 以求解
定義

A:m∗nA:m*nA:m∗n, 若 A→rUA \xrightarrow{r} UAr​U 為 row echelon form, 則 UUU 之非零列個數稱為 AAA 之 rank, 記作 rank(A)rank(A)rank(A) 或 r(A)r(A)r(A)

  • UUU 之非零列個數 = pivot 個數
定理

A:m∗nA:m*nA:m∗n, x⃗:n∗1\vec{x}:n*1x:n∗1, b⃗:m∗1\vec{b}:m*1b:m∗1, 則:

  • r(A)≠r([A∣b⃗])⇔Ax⃗=b⃗r(A) \ne r([A|\vec{b}]) \Leftrightarrow A\vec{x}=\vec{b}r(A)​=r([A∣b])⇔Ax=b 無解
  • r(A)=r([A∣b⃗])=n⇔Ax⃗=b⃗r(A)=r([A|\vec{b}])=n \Leftrightarrow A\vec{x}=\vec{b}r(A)=r([A∣b])=n⇔Ax=b 具唯一解
  • r(A)=r([A∣b⃗])<n⇔Ax⃗=b⃗r(A)=r([A|\vec{b}])<n \Leftrightarrow A\vec{x}=\vec{b}r(A)=r([A∣b])<n⇔Ax=b 具無限多解
    • 其中 n−rank(A)n-rank(A)n−rank(A) 為自由變數之個數

可逆之充要條件

可逆的等價命題

引理

A:n∗nA:n*nA:n∗n, Ax⃗=0⃗A\vec{x}=\vec{0}Ax=0 只有零解 ⇔A∼rIn\Leftrightarrow A \overset{r}{\sim} I_n⇔A∼rIn​

定理

A:n∗nA:n*nA:n∗n, 則以下敘述等價:

  • AAA 可逆
  • Ax⃗=0⃗A\vec{x}=\vec{0}Ax=0 只有零解 (nonsingular)
  • A∼rInA \overset{r}{\sim} I_nA∼rIn​
  • AAA 可以寫成列基本矩陣的乘積
Note

反矩陣的求法:
若 A:n∗nA:n*nA:n∗n 可逆, 則 [A∣In]→r[In∣A−1][A|I_n] \xrightarrow{r} [I_n|A^{-1}][A∣In​]r​[In​∣A−1]

定理

A:n∗nA:n*nA:n∗n, 則以下敘述等價:

  • AAA 可逆
  • ∀b⃗:n∗1\forall \vec{b}:n*1∀b:n∗1, Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b 具有唯一解
  • ∀b⃗:n∗1\forall \vec{b}:n*1∀b:n∗1, Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b 有解

LU 分解

定義
  • A=LUA=LUA=LU: 稱為 LU decomposition
    • LLL 為 lower triangular matrix
    • UUU 為 upper triangular matrix
  • 若將 UUU 的非零列首項提出為對角矩陣 DDD, 可得 A=LDUA=LDUA=LDU, 稱為 LDU decomposition
Note
  • 並非每個矩陣皆可作 LU decomposition
  • LU decomposition 通常不允許列交換, 因此高斯消去法只能用上面列加下面列的方式進行
  • LU decomposition 的目的: 省去高斯消去法花時間的列運算
Note

求解 Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b

  • 若 A=LUA=LUA=LU, 則 Ax⃗=b⃗⇒L(Ux⃗)=b⃗A\vec{x}=\vec{b} \Rightarrow L(U\vec{x})=\vec{b}Ax=b⇒L(Ux)=b
  • 令 y⃗=Ux⃗\vec{y}=U\vec{x}y​=Ux, 先解 Ly⃗=b⃗L\vec{y}=\vec{b}Ly​=b 求 y⃗\vec{y}y​ (forward substitution)
  • 再解 Ux⃗=y⃗U\vec{x}=\vec{y}Ux=y​ 求 x⃗\vec{x}x (back substitution)

行運算

Note
  • A→cBA \xrightarrow{c} BAc​B (行運算 ccc 將 AAA 變為 BBB), 且 E=c(I)E=c(I)E=c(I), 則 B=AEB=AEB=AE
  • A∼cBA \overset{c}{\sim} BA∼cB (AAA 行等價於 BBB) ⇔∃Q\Leftrightarrow \exists Q⇔∃Q 為可逆矩陣, 使得 B=AQB=AQB=AQ
定理

A:n∗nA:n*nA:n∗n, 則以下敘述等價:

  • AAA 可逆
  • x⃗A=0⃗\vec{x}A=\vec{0}xA=0 只有零解
  • A∼cInA \overset{c}{\sim} I_nA∼cIn​
  • AAA 可以寫成行基本矩陣的乘積
  • ∀b⃗:1∗n\forall \vec{b}:1*n∀b:1∗n, x⃗A=b⃗\vec{x}A=\vec{b}xA=b 具有唯一解
  • ∀b⃗:1∗n\forall \vec{b}:1*n∀b:1∗n, x⃗A=b⃗\vec{x}A=\vec{b}xA=b 有解
Note
  • 基本矩陣同時是行基本矩陣與列基本矩陣
  • 基本矩陣皆可逆, 且其反矩陣與轉置仍是基本矩陣
  • 基本矩陣相乘不一定是基本矩陣
  • 可逆矩陣不一定是基本矩陣, 但可寫成基本矩陣的乘積
