線性轉換

定義

令 V,V′V,V'V,V′ 為 field FFF 上的 vector spaces，且 T:V→V′T:V\to V'T:V→V′ 為 function。若 TTT 滿足:

  • 對任意 u⃗,v⃗∈V\vec{u},\vec{v}\in Vu,v∈V，有 T(u⃗+v⃗)=T(u⃗)+T(v⃗)T(\vec{u}+\vec{v})=T(\vec{u})+T(\vec{v})T(u+v)=T(u)+T(v)
  • 對任意 c∈Fc\in Fc∈F、v⃗∈V\vec{v}\in Vv∈V，有 T(cv⃗)=cT(v⃗)T(c\vec{v})=cT(\vec{v})T(cv)=cT(v)

則稱 TTT 為 VVV 至 V′V'V′ 之一 linear transformation，也稱為 linear mapping。

定理

T:V→V′T:V\to V'T:V→V′ 為 linear transformation，等價於以下任一條件:

  • 對任意 c,d∈Fc,d\in Fc,d∈F、u⃗,v⃗∈V\vec{u},\vec{v}\in Vu,v∈V，有 T(cu⃗+dv⃗)=cT(u⃗)+dT(v⃗)T(c\vec{u}+d\vec{v})=cT(\vec{u})+dT(\vec{v})T(cu+dv)=cT(u)+dT(v)
  • 對任意 c∈Fc\in Fc∈F、u⃗,v⃗∈V\vec{u},\vec{v}\in Vu,v∈V，有 T(cu⃗+v⃗)=cT(u⃗)+T(v⃗)T(c\vec{u}+\vec{v})=cT(\vec{u})+T(\vec{v})T(cu+v)=cT(u)+T(v)
Note

若 T:V→V′T:V\to V'T:V→V′ 為 linear transformation，則:

  • 重要 T(0⃗)=0⃗T(\vec{0})=\vec{0}T(0)=0
  • T(−v⃗)=−T(v⃗)T(-\vec{v})=-T(\vec{v})T(−v)=−T(v)
  • T(u⃗−v⃗)=T(u⃗)−T(v⃗)T(\vec{u}-\vec{v})=T(\vec{u})-T(\vec{v})T(u−v)=T(u)−T(v)
定理

令 V,V′V,V'V,V′ 為 field FFF 上的 vector spaces，且 β={v⃗1,…,v⃗n}\beta=\{\vec{v}_1,\dots,\vec{v}_n\}β={v1​,…,vn​} 為 VVV 的一組 basis。

令 T:V→V′T:V\to V'T:V→V′ 為 linear transformation。

  • TTT is 1-1   ⟺  ker⁡(T)={0⃗}\iff \ker(T)=\{\vec{0}\}⟺ker(T)={0}   ⟺  nullity⁡(T)=0\iff \operatorname{nullity}(T)=0⟺nullity(T)=0   ⟺  rank⁡(T)=dim⁡(V)\iff \operatorname{rank}(T)=\dim(V)⟺rank(T)=dim(V)。
  • TTT is onto   ⟺  R(T)=V′\iff R(T)=V'⟺R(T)=V′   ⟺  rank⁡(T)=dim⁡(V′)\iff \operatorname{rank}(T)=\dim(V')⟺rank(T)=dim(V′)   ⟺  nullity⁡(T)=dim⁡(V)−dim⁡(V′)\iff \operatorname{nullity}(T)=\dim(V)-\dim(V')⟺nullity(T)=dim(V)−dim(V′)。

可得:

  • 若 TTT is 1-1，則 dim⁡(V)≤dim⁡(V′)\dim(V)\le \dim(V')dim(V)≤dim(V′)。
  • 若 TTT is onto，則 dim⁡(V)≥dim⁡(V′)\dim(V)\ge \dim(V')dim(V)≥dim(V′)。
  • 若 TTT is 1-1 且 onto，則 dim⁡(V)=dim⁡(V′)\dim(V)=\dim(V')dim(V)=dim(V′)。
  • 對有限維向量空間而言，同構必同維。
Note

矩陣轉換與四大子空間:

令 AAA 為 m×nm\times nm×n matrix，定義 T:Rn→RmT:\mathbb{R}^n\to\mathbb{R}^mT:Rn→Rm by T(x⃗)=Ax⃗T(\vec{x})=A\vec{x}T(x)=Ax，則 TTT 為 linear transformation。

  • R(T)=Im⁡(T)={Ax⃗∣x⃗∈Rn}=CS(A)=R(A)R(T)=\operatorname{Im}(T)=\{A\vec{x}\mid \vec{x}\in\mathbb{R}^n\}=CS(A)=R(A)R(T)=Im(T)={Ax∣x∈Rn}=CS(A)=R(A)。
  • N(T)=ker⁡(T)={x⃗∈Rn∣Ax⃗=0⃗}=ker⁡(A)=N(A)N(T)=\ker(T)=\{\vec{x}\in\mathbb{R}^n\mid A\vec{x}=\vec{0}\}=\ker(A)=N(A)N(T)=ker(T)={x∈Rn∣Ax=0}=ker(A)=N(A)。
  • RS(A)=R(AT)RS(A)=R(A^T)RS(A)=R(AT)。
  • Lker⁡(A)=ker⁡(AT)=N(AT)\operatorname{Lker}(A)=\ker(A^T)=N(A^T)Lker(A)=ker(AT)=N(AT)。

因此:

  • TTT is 1-1   ⟺  ker⁡(T)=ker⁡(A)={0⃗}\iff \ker(T)=\ker(A)=\{\vec{0}\}⟺ker(T)=ker(A)={0}   ⟺  A\iff A⟺A 的欄向量 LI   ⟺  rank⁡(A)=n\iff \operatorname{rank}(A)=n⟺rank(A)=n。
    • 必為長矩陣 (反向不成立)
  • TTT is onto   ⟺  R(T)=CS(A)=Rm\iff R(T)=CS(A)=\mathbb{R}^m⟺R(T)=CS(A)=Rm   ⟺  A\iff A⟺A 的欄向量生成 Rm\mathbb{R}^mRm   ⟺  rank⁡(A)=m\iff \operatorname{rank}(A)=m⟺rank(A)=m。
    • 必為扁矩陣 (反向不成立)

所以 AAA 若同時滿足欄向量 LI 且生成 Rm\mathbb{R}^mRm，則 TTT 為 bijective。

定理

矩陣的 Rank

定義

令 AAA 為 m×nm\times nm×n matrix。

  • 定義 rr(A)=dim⁡(RS(A))rr(A)=\dim(RS(A))rr(A)=dim(RS(A))，稱為 AAA 的 row rank。
  • 定義 cr(A)=dim⁡(CS(A))cr(A)=\dim(CS(A))cr(A)=dim(CS(A))，稱為 AAA 的 column rank。
定理

令 AAA 為 m×nm\times nm×n matrix，則 rr(A)=cr(A)rr(A)=cr(A)rr(A)=cr(A)

共同的值稱為 AAA 的 rank，記作 rank⁡(A)\operatorname{rank}(A)rank(A)。
可從三個角度理解:

  • row space 的維度。
  • column space 的維度。
  • 將 AAA 化為 rref 後，非零列的數目。
Note
  • A=O  ⟺  rank⁡(A)=0A=O \iff \operatorname{rank}(A)=0A=O⟺rank(A)=0。
  • 若 AAA 為 m×nm\times nm×n matrix，則 rank⁡(A)≤min⁡{m,n}\operatorname{rank}(A)\le \min\{m,n\}rank(A)≤min{m,n}。
  • rank⁡(A)=rank⁡(AT)\operatorname{rank}(A)=\operatorname{rank}(A^T)rank(A)=rank(AT)。
  • 若 A∼rBA\overset{r}{\sim}BA∼rB，則 rank⁡(A)=rank⁡(B)\operatorname{rank}(A)=\operatorname{rank}(B)rank(A)=rank(B)。
  • 若 A∼cBA\overset{c}{\sim}BA∼cB，則 rank⁡(A)=rank⁡(B)\operatorname{rank}(A)=\operatorname{rank}(B)rank(A)=rank(B)。
  • 若 P,QP,QP,Q 皆可逆，則 rank⁡(PA)=rank⁡(A)\operatorname{rank}(PA)=\operatorname{rank}(A)rank(PA)=rank(A)，且 rank⁡(AQ)=rank⁡(A)\operatorname{rank}(AQ)=\operatorname{rank}(A)rank(AQ)=rank(A)。
定理

令 AAA 為 m×nm\times nm×n matrix，BBB 為 n×pn\times pn×p matrix，則 rank⁡(AB)≤min⁡{rank⁡(A),rank⁡(B)}\operatorname{rank}(AB)\le \min\{\operatorname{rank}(A),\operatorname{rank}(B)\}rank(AB)≤min{rank(A),rank(B)}

範例

令 AAA 為 m×nm\times nm×n matrix，BBB 為 n×mn\times mn×m matrix，且 m>nm>nm>n。證明 ABABAB 不可逆。
因為 ABABAB 為 m×mm\times mm×m matrix，且 rank⁡(AB)≤rank⁡(A)≤n<m\operatorname{rank}(AB)\le \operatorname{rank}(A)\le n<mrank(AB)≤rank(A)≤n<m，故 ABABAB 不可逆。

Note

若 AAA 為非零矩陣，則 rank⁡(A)=1\operatorname{rank}(A)=1rank(A)=1 等價於 AAA 可寫成 A=x⃗y⃗TA=\vec{x}\vec{y}^{T}A=xy​T

例如:

Note

rank 的幾何意義: 矩陣可視為線性轉換。

令 AAA 為 m×nm\times nm×n matrix，並定義 T:Rn→RmT:\mathbb{R}^{n}\to\mathbb{R}^{m}T:Rn→Rm by T(x⃗)=Ax⃗T(\vec{x})=A\vec{x}T(x)=Ax。則:

  • rank⁡(T)=dim⁡(R(T))=dim⁡(R(A))=rank⁡(A)\operatorname{rank}(T)=\dim(R(T))=\dim(R(A))=\operatorname{rank}(A)rank(T)=dim(R(T))=dim(R(A))=rank(A)。
  • nullity⁡(T)=dim⁡(N(T))=dim⁡(N(A))=nullity⁡(A)\operatorname{nullity}(T)=\dim(N(T))=\dim(N(A))=\operatorname{nullity}(A)nullity(T)=dim(N(T))=dim(N(A))=nullity(A)。
定理

矩陣版維度定理:

令 AAA 為 m×nm\times nm×n matrix，則 rank⁡(A)+nullity⁡(A)=n\operatorname{rank}(A)+\operatorname{nullity}(A)=nrank(A)+nullity(A)=n

  • nnn 是 AAA 的欄數，也就是對應線性轉換 T:Rn→RmT:\mathbb{R}^{n}\to\mathbb{R}^{m}T:Rn→Rm 的定義域維度。
Lemma

令 A,BA,BA,B 皆為 m×nm\times nm×n matrix，則:

  • CS(A+B)⊆CS(A)+CS(B)CS(A+B)\subseteq CS(A)+CS(B)CS(A+B)⊆CS(A)+CS(B)。
  • RS(A+B)⊆RS(A)+RS(B)RS(A+B)\subseteq RS(A)+RS(B)RS(A+B)⊆RS(A)+RS(B)。
定理

令 A,BA,BA,B 皆為 m×nm\times nm×n matrix，則 rank⁡(A+B)≤rank⁡(A)+rank⁡(B)\operatorname{rank}(A+B)\le \operatorname{rank}(A)+\operatorname{rank}(B)rank(A+B)≤rank(A)+rank(B)

Corollary

令 A,BA,BA,B 皆為 m×nm\times nm×n matrix，則 rank⁡(A−B)≥rank⁡(A)−rank⁡(B)\operatorname{rank}(A-B)\ge \operatorname{rank}(A)-\operatorname{rank}(B)rank(A−B)≥rank(A)−rank(B)

  • rank⁡(A)≤rank⁡(A−B)+rank⁡(B)\operatorname{rank}(A)\le \operatorname{rank}(A-B)+\operatorname{rank}(B)rank(A)≤rank(A−B)+rank(B)
定理

令 AAA 為 m×nm\times nm×n matrix，則:

  • AAA 具有左反矩陣   ⟺  rank⁡(A)=n≤m\iff \operatorname{rank}(A)=n\le m⟺rank(A)=n≤m。
  • AAA 具有右反矩陣   ⟺  rank⁡(A)=m≤n\iff \operatorname{rank}(A)=m\le n⟺rank(A)=m≤n。
Note

Note

令 AAA 為 m×nm\times nm×n matrix，x⃗∈Fn×1\vec{x}\in F^{n\times 1}x∈Fn×1，b⃗∈Fm×1\vec{b}\in F^{m\times 1}b∈Fm×1。

  • Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b 有解   ⟺  b⃗∈CS(A)\iff \vec{b}\in CS(A)⟺b∈CS(A)。
  • Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b 對所有 b⃗∈Fm×1\vec{b}\in F^{m\times 1}b∈Fm×1 皆有解  ⟺  CS(A)=Fm×1  ⟺  rank⁡(A)=m\iff CS(A)=F^{m\times 1}\iff \operatorname{rank}(A)=m⟺CS(A)=Fm×1⟺rank(A)=m
    • 欄向量生成定義域
  • Ax⃗=b⃗A\vec{x}=\vec{b}Ax=b 對所有 b⃗\vec{b}b 至多一解   ⟺  rank⁡(A)=n\iff \operatorname{rank}(A)=n⟺rank(A)=n
