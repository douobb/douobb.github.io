Ch4 線性映射

線性轉換

定義

令 $V,V’$ 為 field $F$ 上的 vector spaces，且 $T:V\to V’$ 為 function。若 $T$ 滿足:

  • 對任意 $\vec{u},\vec{v}\in V$，有 $T(\vec{u}+\vec{v})=T(\vec{u})+T(\vec{v})$
  • 對任意 $c\in F$、$\vec{v}\in V$，有 $T(c\vec{v})=cT(\vec{v})$

則稱 $T$ 為 $V$ 至 $V’$ 之一 linear transformation，也稱為 linear mapping。

定理

$T:V\to V’$ 為 linear transformation，等價於以下任一條件:

  • 對任意 $c,d\in F$、$\vec{u},\vec{v}\in V$，有 $T(c\vec{u}+d\vec{v})=cT(\vec{u})+dT(\vec{v})$
  • 對任意 $c\in F$、$\vec{u},\vec{v}\in V$，有 $T(c\vec{u}+\vec{v})=cT(\vec{u})+T(\vec{v})$
Note

若 $T:V\to V’$ 為 linear transformation，則:

  • 重要 $T(\vec{0})=\vec{0}$
  • $T(-\vec{v})=-T(\vec{v})$
  • $T(\vec{u}-\vec{v})=T(\vec{u})-T(\vec{v})$
定理

令 $V,V’$ 為 field $F$ 上的 vector spaces，且 $\beta={\vec{v}_1,\dots,\vec{v}_n}$ 為 $V$ 的一組 basis。

若任取 $\vec{w}_1,\dots,\vec{w}_n\in V’$，則存在唯一的 linear transformation $T:V\to V’$，使得 $T(\vec{v}_1)=\vec{w}_1,\dots,T(\vec{v}_n)=\vec{w}_n$

Note

matrix transformation:
令 $A$ 為 $m\times n$ matrix，定義 $T:F^{n\times 1}\to F^{m\times 1}$ by $T(\vec{x})=A\vec{x}$ 則 $T$ 為 linear transformation。

Note

若 $T:\mathbb{R}^n\to\mathbb{R}^m$ 為 linear transformation，則存在唯一 $m\times n$ matrix $A$，使得 $T(\vec{x})=A\vec{x}$

其中 $A=[T(\vec{e}_1)\ T(\vec{e}_2)\ \cdots\ T(\vec{e}_n)]$為 $T$ 的 standard matrix。

Note

常見之 linear transformation:

  • rotation: 旋轉
  • projection: 投影
  • reflection: 鏡射

座標化

定義

令 $\beta={\vec{v}_1,\vec{v}_2,\dots,\vec{v}_n}$ 為 $V$ 的一組 ordered basis。

若 $\vec{v}\in V$，且 $\vec{v}=c_1\vec{v}_1+c_2\vec{v}_2+\cdots+c_n\vec{v}_n$

則稱

為 $\vec{v}$ 相對於 $\beta$ 之 coordinate vector。

Note

設 $\beta$ 為 $V$ 的一組 ordered basis。

  1. $[\vec{v}]_{\beta}$ 存在且唯一。
  2. $[c1\vec{v}_1+\cdots+c_k\vec{v}_k]{\beta}=c1[\vec{v}_1]{\beta}+\cdots+ck[\vec{v}_k]{\beta}$。
  3. $[\vec{u}]{\beta}=[\vec{v}]{\beta}\iff \vec{u}=\vec{v}$。
  4. 若 $\dim(V)=n$，則對任意 $\vec{x}\in F^{n\times1}$，存在唯一 $\vec{v}\in V$，使得 $[\vec{v}]_{\beta}=\vec{x}$。
定義

令 $V,W$ 為 field $F$ 上的 vector spaces。
若存在 $T:V\to W$，且 $T$ 為 linear、one-to-one、onto，則稱 $T$ 為 linear isomorphism (線性同構函數)。

此時稱 $V$ 與 $W$ 為 isomorphic (同構)，記作 $V\cong W$

Note

有限維向量空間的同構 (維度相同即同構):

  • 若 $\dim(V)=n$，則 $V\cong F^{n\times1}$。
  • $\cong$ 為一種等價關係。
  • 若 $\dim(V)=\dim(W)=n$，則 $V\cong W$。
Note

令 $\beta={\vec{v}_1,\dots,\vec{v}_n}$ 與 $\gamma={\vec{u}_1,\dots,\vec{u}_n}$ 皆為 $V$ 的 ordered basis。

若 $\vec{v}\in V$，且 $\vec{v}=c_1\vec{v}_1+\cdots+c_n\vec{v}_n$

將 $\vec{v}$ 改以 $\gamma$ 表示時，有 $[\vec{v}]{\gamma}=[c_1\vec{v}_1+\cdots+c_n\vec{v}_n]{\gamma}=c1[\vec{v}_1]{\gamma}+\cdots+cn[\vec{v}_n]{\gamma}$

因此 $[\vec{v}]{\gamma}=\begin{bmatrix}[\vec{v}_1]{\gamma}&[\vec{v}2]{\gamma}&\cdots&[\vec{v}n]{\gamma}\end{bmatrix}[\vec{v}]{\beta}=[I]{\beta}^{\gamma}[\vec{v}]_{\beta}$

稱 $[I]_{\beta}^{\gamma}$ 為從 $\beta$ 到 $\gamma$ 的 transition matrix，或 change of coordinate matrix。

  • $[I]{\beta}^{\gamma}$ 可逆，且 $[I]{\gamma}^{\beta}=\left([I]_{\beta}^{\gamma}\right)^{-1}$
Note

求 transition matrix 時，可使用標準基底作為中間項。
令 $\delta$ 為 $F^{n\times1}$ 的 standard basis，則 $[I]{\beta}^{\gamma}=[I]{\delta}^{\gamma}[I]{\beta}^{\delta} =\left([I]{\gamma}^{\delta}\right)^{-1}[I]_{\beta}^{\delta}$

  • 可利用 $[B\mid A]\overset{r}{\sim}[I\mid B^{-1}A]$ 求解

設 $\beta={\vec{v}_1,\dots,\vec{v}_n}$ 為 $V$ 的 ordered basis，$\gamma={\vec{u}_1,\dots,\vec{u}_m}$ 為 $V’$ 的 ordered basis。

若 $\vec{v}\in V$，且 $\vec{v}=c_1\vec{v}_1+\cdots+c_n\vec{v}_n$，則

由線性性質可得 $[T(\vec{v})]{\gamma}=[T(c_1\vec{v}_1+\cdots+c_n\vec{v}_n)]{\gamma}=c1[T(\vec{v}_1)]{\gamma}+\cdots+cn[T(\vec{v}_n)]{\gamma}$

Kernel 與 Image

定義

令 $T:V\to V’$ 為 linear transformation。

  • 若 $S\subseteq V$，定義 $T(S)={T(\vec{v})\mid \vec{v}\in S}$，稱為 $S$ under $T$ 的 direct image。
  • 若 $S’\subseteq V’$，定義 $T^{-1}(S’)={\vec{v}\in V\mid T(\vec{v})\in S’}$，稱為 $S’$ under $T$ 的 inverse image。
    • $T^{-1}(S’)$ 不需要 $T$ 可逆。
定理

保持 subspace:
令 $T:V\to V’$ 為 linear transformation。

  • 若 $W\overset{s}{\subseteq}V$，則 $T(W)\overset{s}{\subseteq}V’$。
  • 若 $W’\overset{s}{\subseteq}V’$，則 $T^{-1}(W’)\overset{s}{\subseteq}V$。
定義

令 $T:V\to V’$ 為 linear transformation。

  • $\ker(T)={\vec{v}\in V\mid T(\vec{v})=\vec{0}}$，稱為 $T$ 的 kernel/nullspace，記作 $N(T)$。
  • $\operatorname{Im}(T)=T(V)={T(\vec{v})\mid \vec{v}\in V}$，稱為 $T$ 的 image/range，記作 $R(T)$。
Note
  1. 因為 $V\overset{s}{\subseteq}V$，所以 $\operatorname{Im}(T)=T(V)\overset{s}{\subseteq}V’$。
  2. 因為 ${\vec{0}}\overset{s}{\subseteq}V’$，所以 $\ker(T)=T^{-1}({\vec{0}})\overset{s}{\subseteq}V$。
  3. 定義 $\operatorname{rank}(T)=\dim(\operatorname{Im}(T))$。
  4. 定義 $\operatorname{nullity}(T)=\dim(\ker(T))$。
定理

Rank-Nullity Theorem / Dimension Theorem / Sylvester's 1st Law:

令 $T:V\to V’$ 為 linear transformation。則

Note

令 $T:V\to V’$ 為 linear transformation。

  • $T$ 為 1-1/injective $\iff$ 若 $T(\vec{u})=T(\vec{v})$，則 $\vec{u}=\vec{v}$。
  • $T$ 為 onto/surjective $\iff R(T)=V’$。
  • $T$ 為 bijective $\iff T$ 同時為 1-1 且 onto。
定理

令 $T:V\to V’$ 為 linear transformation。

則 $T\text{ is 1-1}\iff \ker(T)={\vec{0}}$

Note

令 $T:V\to V’$ 為 linear transformation。

  • $T$ is 1-1 $\iff \ker(T)={\vec{0}}$ $\iff \operatorname{nullity}(T)=0$ $\iff \operatorname{rank}(T)=\dim(V)$。
  • $T$ is onto $\iff R(T)=V’$ $\iff \operatorname{rank}(T)=\dim(V’)$ $\iff \operatorname{nullity}(T)=\dim(V)-\dim(V’)$。

可得:

  • 若 $T$ is 1-1，則 $\dim(V)\le \dim(V’)$。
  • 若 $T$ is onto，則 $\dim(V)\ge \dim(V’)$。
  • 若 $T$ is 1-1 且 onto，則 $\dim(V)=\dim(V’)$。
  • 對有限維向量空間而言，同構必同維。
Note

矩陣轉換與四大子空間:

令 $A$ 為 $m\times n$ matrix，定義 $T:\mathbb{R}^n\to\mathbb{R}^m$ by $T(\vec{x})=A\vec{x}$，則 $T$ 為 linear transformation。

  • $R(T)=\operatorname{Im}(T)={A\vec{x}\mid \vec{x}\in\mathbb{R}^n}=CS(A)=R(A)$。
  • $N(T)=\ker(T)={\vec{x}\in\mathbb{R}^n\mid A\vec{x}=\vec{0}}=\ker(A)=N(A)$。
  • $RS(A)=R(A^T)$。
  • $\operatorname{Lker}(A)=\ker(A^T)=N(A^T)$。

因此:

  • $T$ is 1-1 $\iff \ker(T)=\ker(A)={\vec{0}}$ $\iff A$ 的欄向量 LI $\iff \operatorname{rank}(A)=n$。
    • 必為長矩陣 (反向不成立)
  • $T$ is onto $\iff R(T)=CS(A)=\mathbb{R}^m$ $\iff A$ 的欄向量生成 $\mathbb{R}^m$ $\iff \operatorname{rank}(A)=m$。
    • 必為扁矩陣 (反向不成立)

所以 $A$ 若同時滿足欄向量 LI 且生成 $\mathbb{R}^m$，則 $T$ 為 bijective。
定理
令 $T:V\to V’$ 為 linear transformation。
若 $\dim(V)=\dim(V’)$，則 $T\text{ is 1-1}\iff T\text{ is onto}$

定義

令 $T:V\to V’$ 為 linear transformation。

  1. 若對任意 $S\subseteq V$，$S$ 為 LD 時皆有 $T(S)$ 為 LD，則稱 $T$ 保相依。
  2. 若對任意 $S\subseteq V$，$S$ 為 LI 時皆有 $T(S)$ 為 LI，則稱 $T$ 保獨立。
  3. 若對任意 $S\subseteq V$，$S$ 生成 $V$ 時皆有 $T(S)$ 生成 $V’$，則稱 $T$ 保生成。
定理

令 $T:V\to V’$ 為 linear transformation。
則 $T$ 保相依。

定理

令 $T:V\to V’$ 為 linear transformation。
則 $T$ is 1-1 $\iff T$ 保獨立。

Note
  • 對任意 linear transformation，若 ${\vec{v}_1,\dots,\vec{v}_k}$ 為 LD，則 ${T(\vec{v}_1),\dots,T(\vec{v}_k)}$ 為 LD。
  • 若 $T$ is 1-1，則 ${\vec{v}_1,\dots,\vec{v}_k}$ 為 LI $\iff {T(\vec{v}_1),\dots,T(\vec{v}_k)}$ 為 LI。
  • 若 $T$ is 1-1，則 ${\vec{v}_1,\dots,\vec{v}_k}$ 為 LD $\iff {T(\vec{v}_1),\dots,T(\vec{v}_k)}$ 為 LD。
定理

令 $T:V\to V’$ 為 linear transformation。
則 $T$ is onto $\iff T$ 保生成。

矩陣的 Rank

定義

令 $A$ 為 $m\times n$ matrix。

  • 定義 $rr(A)=\dim(RS(A))$，稱為 $A$ 的 row rank。
  • 定義 $cr(A)=\dim(CS(A))$，稱為 $A$ 的 column rank。
定理

令 $A$ 為 $m\times n$ matrix，則 $rr(A)=cr(A)$

共同的值稱為 $A$ 的 rank，記作 $\operatorname{rank}(A)$。
可從三個角度理解:

  • row space 的維度。
  • column space 的維度。
  • 將 $A$ 化為 rref 後，非零列的數目。
Note
  • $A=O \iff \operatorname{rank}(A)=0$。
  • 若 $A$ 為 $m\times n$ matrix，則 $\operatorname{rank}(A)\le \min{m,n}$。
  • $\operatorname{rank}(A)=\operatorname{rank}(A^T)$。
  • 若 $A\overset{r}{\sim}B$，則 $\operatorname{rank}(A)=\operatorname{rank}(B)$。
  • 若 $A\overset{c}{\sim}B$，則 $\operatorname{rank}(A)=\operatorname{rank}(B)$。
  • 若 $P,Q$ 皆可逆，則 $\operatorname{rank}(PA)=\operatorname{rank}(A)$，且 $\operatorname{rank}(AQ)=\operatorname{rank}(A)$。
定理

令 $A$ 為 $m\times n$ matrix，$B$ 為 $n\times p$ matrix，則 $\operatorname{rank}(AB)\le \min{\operatorname{rank}(A),\operatorname{rank}(B)}$

範例

令 $A$ 為 $m\times n$ matrix，$B$ 為 $n\times m$ matrix，且 $m>n$。證明 $AB$ 不可逆。
因為 $AB$ 為 $m\times m$ matrix，且 $\operatorname{rank}(AB)\le \operatorname{rank}(A)\le n<m$，故 $AB$ 不可逆。

Note

若 $A$ 為非零矩陣，則 $\operatorname{rank}(A)=1$ 等價於 $A$ 可寫成 $A=\vec{x}\vec{y}^{T}$

例如:

因此 $A=\vec{x}\vec{y}^{T}$，且 $\operatorname{rank}(A)=1$。

Note

rank 的幾何意義: 矩陣可視為線性轉換。

令 $A$ 為 $m\times n$ matrix，並定義 $T:\mathbb{R}^{n}\to\mathbb{R}^{m}$ by $T(\vec{x})=A\vec{x}$。則:

  • $\operatorname{rank}(T)=\dim(R(T))=\dim(R(A))=\operatorname{rank}(A)$。
  • $\operatorname{nullity}(T)=\dim(N(T))=\dim(N(A))=\operatorname{nullity}(A)$。
定理

矩陣版維度定理:

令 $A$ 為 $m\times n$ matrix，則 $\operatorname{rank}(A)+\operatorname{nullity}(A)=n$

  • $n$ 是 $A$ 的欄數，也就是對應線性轉換 $T:\mathbb{R}^{n}\to\mathbb{R}^{m}$ 的定義域維度。
Lemma

令 $A,B$ 皆為 $m\times n$ matrix，則:

  • $CS(A+B)\subseteq CS(A)+CS(B)$。
  • $RS(A+B)\subseteq RS(A)+RS(B)$。
定理

令 $A,B$ 皆為 $m\times n$ matrix，則 $\operatorname{rank}(A+B)\le \operatorname{rank}(A)+\operatorname{rank}(B)$

Corollary

令 $A,B$ 皆為 $m\times n$ matrix，則 $\operatorname{rank}(A-B)\ge \operatorname{rank}(A)-\operatorname{rank}(B)$

  • $\operatorname{rank}(A)\le \operatorname{rank}(A-B)+\operatorname{rank}(B)$
定理

令 $A$ 為 $m\times n$ matrix，則:

  • $A$ 具有左反矩陣 $\iff \operatorname{rank}(A)=n\le m$。
  • $A$ 具有右反矩陣 $\iff \operatorname{rank}(A)=m\le n$。
Note

令 $A$ 為 $m\times n$ matrix，則

$A$ 具有左反矩陣 $\iff \operatorname{rank}(A)=n$
$\iff \dim(CS(A))=n\iff A \text{ 的欄向量 LI in } F^{m\times 1}$
$\iff \dim(RS(A))=n\iff A \text{ 的列向量生成 } F^{1\times n}$

$A$ 具有右反矩陣$\iff \operatorname{rank}(A)=m$
$\iff \dim(RS(A))=m\iff A \text{ 的列向量 LI in } F^{1\times n}$
$\iff \dim(CS(A))=m\iff A \text{ 的欄向量生成 } F^{m\times 1}$

Note

令 $A$ 為 $m\times n$ matrix，$\vec{x}\in F^{n\times 1}$，$\vec{b}\in F^{m\times 1}$。

  • $A\vec{x}=\vec{b}$ 有解 $\iff \vec{b}\in CS(A)$。
  • $A\vec{x}=\vec{b}$ 對所有 $\vec{b}\in F^{m\times 1}$ 皆有解$\iff CS(A)=F^{m\times 1}\iff \operatorname{rank}(A)=m$
    • 欄向量生成定義域
  • $A\vec{x}=\vec{b}$ 對所有 $\vec{b}$ 至多一解 $\iff \operatorname{rank}(A)=n$
    • 欄向量 LI
    • 仍可能無解，但有解則唯一
範例

  1. 若 $A$ 為 $3\times 5$ matrix，且 $\operatorname{rank}(A)=3$:
    對所有 $\vec{b}$，$A\vec{x}=\vec{b}$ 皆有解。
    又因 $n-r(A)=5-3=2$，所以每個 $\vec{b}$ 都有無限多解。

  2. 若 $A$ 為 $4\times 3$ matrix，且 $\operatorname{rank}(A)=3$:
    $A$ 的欄向量 LI，所以對所有 $\vec{b}$，$A\vec{x}=\vec{b}$ 至多一解。
    因 $r(A)<m$，並非每個 $\vec{b}$ 都有解。

  3. 若 $A$ 為 $3\times 4$ matrix，且 $\operatorname{rank}(A)=2$:
    $r(A)<m$，並非每個 $\vec{b}$ 都有解。
    若 $\vec{b}\in CS(A)$，則因 $n-r(A)=4-2=2$，有無限多解。

四大空間之基底

Note

令 $A$ 為 $m\times n$ matrix。

  • $CS(A)=R(A)$
  • $RS(A)=R(A^T)$
  • $\ker(A)=N(A)$
  • $\operatorname{Lker}(A)=N(A^T)$

且有:

  • $\dim(R(A))+\dim(N(A))=n$
  • $\dim(R(A^T))+\dim(N(A^T))=m$
範例

求四大空間基底的流程:

  1. 先將 $A$ 化為 rref，記作 $U$。
  2. $RS(A)$: 取 $U$ 的非零列向量。
  3. $\ker(A)$: 解 $U\vec{x}=\vec{0}$，由一般解取參數向量。
  4. $CS(A)$: 找出 $U$ 的 pivot columns，再回到原矩陣 $A$ 取對應欄向量。
  5. $\operatorname{Lker}(A)$: 解 $A^T\vec{y}=\vec{0}$。

  • 且 $[UT]{\beta}^{\delta}=[U]{\gamma}^{\delta}[T]_{\beta}^{\gamma}$
