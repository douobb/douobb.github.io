Ch5 對角化及其應用

相似

定義

令 A,BA,BA,B 皆為 n×nn\times nn×n matrices。若存在 n×nn\times nn×n 可逆矩陣 PPP，使得 P−1AP=BP^{-1}AP=BP−1AP=B
則稱 BBB is similar to AAA，記作 A∼BA\sim BA∼B。

Note

相似是一種等價關係:

  • 反身性: A∼AA\sim AA∼A，因為 I−1AI=AI^{-1}AI=AI−1AI=A。

  • 對稱性: 若 A∼BA\sim BA∼B，則存在可逆矩陣 PPP 使得 P−1AP=BP^{-1}AP=BP−1AP=B。
    因此 A=PBP−1=(P−1)−1B(P−1)A=PBP^{-1}=(P^{-1})^{-1}B(P^{-1})A=PBP−1=(P−1)−1B(P−1)
    所以 B∼AB\sim AB∼A。

  • 遞移性: 若 A∼BA\sim BA∼B 且 B∼CB\sim CB∼C，則存在可逆矩陣 P,QP,QP,Q 使得 P−1AP=B,Q−1BQ=CP^{-1}AP=B, Q^{-1}BQ=CP−1AP=B,Q−1BQ=C
    因此 C=Q−1BQ=Q−1P−1APQ=(PQ)−1A(PQ)C=Q^{-1}BQ=Q^{-1}P^{-1}APQ=(PQ)^{-1}A(PQ)C=Q−1BQ=Q−1P−1APQ=(PQ)−1A(PQ)
    所以 A∼CA\sim CA∼C。

Note

特殊矩陣的相似關係:

  • A∼O  ⟺  A=OA\sim O \iff A=OA∼O⟺A=O。
  • A∼I  ⟺  A=IA\sim I \iff A=IA∼I⟺A=I。
  • A∼αI  ⟺  A=αIA\sim \alpha I \iff A=\alpha IA∼αI⟺A=αI。
定理

相似變換/對角化下不改變的量:
令 A,BA,BA,B 皆為 n×nn\times nn×n matrices。若 A∼BA\sim BA∼B，則:

  • tr⁡(A)=tr⁡(B)\operatorname{tr}(A)=\operatorname{tr}(B)tr(A)=tr(B)。
  • det⁡(A)=det⁡(B)\det(A)=\det(B)det(A)=det(B)。
  • rank⁡(A)=rank⁡(B)\operatorname{rank}(A)=\operatorname{rank}(B)rank(A)=rank(B)。
  • nullity⁡(A)=nullity⁡(B)\operatorname{nullity}(A)=\operatorname{nullity}(B)nullity(A)=nullity(B)。
定理

若 A∼BA\sim BA∼B，則:

  • Ak∼BkA^k\sim B^kAk∼Bk，其中 k∈Nk\in\mathbb{N}k∈N。
  • cA∼cBcA\sim cBcA∼cB，其中 c∈Fc\in Fc∈F。
  • A+cI∼B+cIA+cI\sim B+cIA+cI∼B+cI，其中 c∈Fc\in Fc∈F。
  • 對任意 polynomial f(x)∈F[x]f(x)\in F[x]f(x)∈F[x]，皆有 f(A)∼f(B)f(A)\sim f(B)f(A)∼f(B)。

對角化

定義

  • 函數版本: 令 T:V→VT:V\to VT:V→V 為 linear operator。若存在 VVV 的一組 basis β\betaβ，使得 [T]ββ=D[T]_{\beta}^{\beta}=D[T]ββ​=D 為對角矩陣，則稱 TTT 可對角化 (diagonalizable)。

  • 矩陣版本: 令 AAA 為 n×nn\times nn×n matrix。若存在可逆矩陣 PPP，使得 P−1AP=DP^{-1}AP=DP−1AP=D (相似) 為對角矩陣，則稱 AAA 可對角化 (diagonalizable)。

定理

令 T:V→VT:V\to VT:V→V 為 linear operator，且 dim⁡(V)=n\operatorname{dim}(V)=ndim(V)=n。

則 TTT 可對角化 ⟺\Longleftrightarrow⟺ TTT 含有 nnn 個 LI eigenvectors。

定理

令 AAA 為 n×nn\times nn×n matrix。

則 AAA 可對角化 ⟺\Longleftrightarrow⟺ AAA 含有 nnn 個 LI eigenvectors。

Note
  • 若 AAA 可對角化，則
    • PPP 的行向量為 AAA 的 eigenvectors
    • DDD 的對角項為對應的 eigenvalues。
  • PPP 不唯一，但 DDD 唯一 (不考慮排列順序)。
  • 若 AAA 不具 nnn 個 LI eigenvectors，則稱 AAA 為 defective matrix，也就是 AAA 不可對角化。

定理

令 T:V→VT:V\to VT:V→V 為 linear operator，且 dim⁡(V)=n\operatorname{dim}(V)=ndim(V)=n。設 λ1,…,λr\lambda_1,\dots,\lambda_rλ1​,…,λr​ 為 TTT 的相異 eigenvalues。

定理

令 AAA 為 n×nn\times nn×n matrix。若 AAA 具有 nnn 個相異 eigenvalues，則 AAA 可對角化。

  • 相異 eigenvalues 對應的 eigenvectors 必為 LI，所以可找到 nnn 個 LI eigenvectors。
定理

冪等算子

定義

  • 令 T:V→VT:V\to VT:V→V 為 linear operator。若 T2=TT^2=TT2=T，則稱 TTT 為 idempotent operator。

  • 令 AAA 為 n×nn\times nn×n matrix。若 A2=AA^2=AA2=A，則稱 AAA 為 idempotent matrix。

Note

對任意 linear operator TTT，有

  • ker⁡(T)⊆ker⁡(T2)⊆ker⁡(T3)⊆⋯\ker(T)\subseteq \ker(T^2)\subseteq \ker(T^3)\subseteq \cdotsker(T)⊆ker(T2)⊆ker(T3)⊆⋯。
  • Im⁡(T)⊇Im⁡(T2)⊇Im⁡(T3)⊇⋯\operatorname{Im}(T)\supseteq \operatorname{Im}(T^2)\supseteq \operatorname{Im}(T^3)\supseteq \cdotsIm(T)⊇Im(T2)⊇Im(T3)⊇⋯。

若 T2=TT^2=TT2=T，則對任意 v⃗∈Im⁡(T)\vec{v}\in \operatorname{Im}(T)v∈Im(T)，可寫成 v⃗=T(u⃗)\vec{v}=T(\vec{u})v=T(u)，因此 T(v⃗)=T2(u⃗)=T(u⃗)=v⃗T(\vec{v})=T^2(\vec{u})=T(\vec{u})=\vec{v}T(v)=T2(u)=T(u)=v。
所以 Im⁡(T)⊆V(1)\operatorname{Im}(T)\subseteq V(1)Im(T)⊆V(1)。反之，若 v⃗∈V(1)\vec{v}\in V(1)v∈V(1)，則 T(v⃗)=v⃗T(\vec{v})=\vec{v}T(v)=v，故 v⃗∈Im⁡(T)\vec{v}\in \operatorname{Im}(T)v∈Im(T)。
因此 V(1)=Im⁡(T)V(1)=\operatorname{Im}(T)V(1)=Im(T)。

Note

若 T2=TT^2=TT2=T，則

  • ker⁡(T)=ker⁡(T2)\ker(T)=\ker(T^2)ker(T)=ker(T2)。
  • nullity⁡(T)=nullity⁡(T2)\operatorname{nullity}(T)=\operatorname{nullity}(T^2)nullity(T)=nullity(T2)。
  • rank⁡(T)=rank⁡(T2)\operatorname{rank}(T)=\operatorname{rank}(T^2)rank(T)=rank(T2)。
  • Im⁡(T)=Im⁡(T2)\operatorname{Im}(T)=\operatorname{Im}(T^2)Im(T)=Im(T2)。
定理

令 T:V→VT:V\to VT:V→V 為 linear operator。則下列敘述等價:

  • V=ker⁡(T)⊕Im⁡(T)V=\ker(T)\oplus \operatorname{Im}(T)V=ker(T)⊕Im(T)。
  • V=ker⁡(T)+Im⁡(T)V=\ker(T)+\operatorname{Im}(T)V=ker(T)+Im(T)。
  • ker⁡(T)∩Im⁡(T)={0⃗}\ker(T)\cap \operatorname{Im}(T)=\{\vec{0}\}ker(T)∩Im(T)={0}。

也就是說，當上述條件成立時，ker⁡(T)\ker(T)ker(T) 與 Im⁡(T)\operatorname{Im}(T)Im(T) 形成 VVV 的 direct sum decomposition。

定理

令 T:V→VT:V\to VT:V→V 為 idempotent operator，即 T2=TT^2=TT2=T。則 V=ker⁡(T)⊕Im⁡(T)V=\ker(T)\oplus \operatorname{Im}(T)V=ker(T)⊕Im(T)

證明

先證 V=ker⁡(T)+Im⁡(T)V=\ker(T)+\operatorname{Im}(T)V=ker(T)+Im(T)。任取 v⃗∈V\vec{v}\in Vv∈V，可寫成 v⃗=(v⃗−T(v⃗))+T(v⃗)\vec{v}=(\vec{v}-T(\vec{v}))+T(\vec{v})v=(v−T(v))+T(v)

其中 T(v⃗)∈Im⁡(T)T(\vec{v})\in \operatorname{Im}(T)T(v)∈Im(T)

且 T(v⃗−T(v⃗))=T(v⃗)−T2(v⃗)=0⃗T(\vec{v}-T(\vec{v}))=T(\vec{v})-T^2(\vec{v})=\vec{0}T(v−T(v))=T(v)−T2(v)=0，故 v⃗−T(v⃗)∈ker⁡(T)\vec{v}-T(\vec{v})\in \ker(T)v−T(v)∈ker(T)。

得到 V=ker⁡(T)⊕Im⁡(T)V=\ker(T)\oplus \operatorname{Im}(T)V=ker(T)⊕Im(T) (by 前一個定理)。

定理

Markov chain

定義

  • stochastic process (隨機過程): 隨時間變化的隨機系統。

  • Markov process: 下一步狀態只與目前狀態有關，與更早以前的狀態無關。

  • Markov matrix/stochastic matrix: entries 皆 ≥0\ge 0≥0，且每一欄和為 111 的矩陣，也稱 transition matrix。

  • probability vector (機率向量): entries 皆 ≥0\ge 0≥0，且 entries 總和為 111 的向量。

Note

  • 若 x⃗0\vec{x}_0x0​ 為 initial probability vector，則 x⃗n=Ax⃗n−1=Anx⃗0\vec{x}_n=A\vec{x}_{n-1}=A^n\vec{x}_0xn​=Axn−1​=Anx0​

    • x⃗n\vec{x}_nxn​ 為第 nnn 個 generation 的 probability vector。

  • 若 probability vector x⃗\vec{x}x 滿足 Ax⃗=x⃗A\vec{x}=\vec{x}Ax=x，則稱 x⃗\vec{x}x 為 steady-state vector，也就是 x⃗∈ker⁡(A−I)\vec{x}\in\ker(A-I)x∈ker(A−I)。

  • 若 AAA 為 transition matrix，且存在 k∈Z+k\in\mathbb{Z}^+k∈Z+ 使 AkA^kAk 中每一項皆為正，則稱 AAA 為 regular transition matrix。
