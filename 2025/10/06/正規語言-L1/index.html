Formal Definition of Finite Automata

Definition

A finite automaton is a 5-tuple (Q,Σ,δ,q0,F)(Q, Σ, δ, q_0, F)(Q,Σ,δ,q0​,F), where

  1. QQQ: is a finite set called the states;
  2. ΣΣΣ: is a finite set called the alphabet;
  3. δδδ: Q×Σ→QQ × Σ → QQ×Σ→Q is the transition function;
  4. q0∈Qq_0 ∈ Qq0​∈Q: is the start state;
  5. F⊆QF ⊆ QF⊆Q: is the set of accept states or final states.

Nondeterminism

  • Deterministic computation: Determined next move.
  • Nondeterministic computation: Several choices may exist for the next move. (類似 fork a process)

Pumping Lemma

Theorem

If AAA is a regular language, then there is a number p (the pumping length) where, if sss is any string in AAA of length at least ppp, then sss may be divided into three pieces, s=xyzs = xyzs=xyz, satisfying the following conditions:

  1. for each i≥0i ≥ 0i≥0, xyiz∈Axy^iz ∈ Axyiz∈A;
  2. ∣y∣>0|y| > 0∣y∣>0; y 不為空字串
  3. ∣xy∣≤p|xy| ≤ p∣xy∣≤p. 在 p 範圍內一定會有重複
  • 證明: 將 AAA 以 DFA 表示，並且此 DFA 的 states 數量為 p，如果一個 string sss 的長度大於 p，則其至少需要經過 p + 1 個 states，根據鴿籠原理必定有 state 是重複經過的，則重複的 state 之間那段就是 y，可以對 y 進行 Pumping 且保持為 regular language

注意:

  • regular language 一定滿足 Pumping Lemma
  • Pumping Lemma 是 regular language 的必要條件，不是充分條件
    • 如果一個 language 滿足 Pumping Lemma，其不一定為 regular language

有了 Pumping Lemma，我們可以利用反證法去證明一個 Languages 是 nonregular languages:

  1. Assume the language, say, BBB, is regular in order to obtain a contradiction.
  2. By the pumping lemma, a pumping length p exists, and any string w∈Bw ∈ Bw∈B can be pumped if ∣w∣≥p|w| ≥ p∣w∣≥p.
  3. Find a string s∈Bs ∈ Bs∈B, ∣s∣≥p|s| ≥ p∣s∣≥p, that sss cannot be pumped as described in the pumping lemma. 尋找反例
  4. The contradiction is obtained, and therefore, BBB is proved to be nonregular.

Myhill-Nerode Theorem

首先定義 indistinguishable 這個關係，主要可以用來將 string 分組:
Relation defined by a given language: Let xxx and yyy be strings and LLL be any language. We say that xxx and yyy are distinguishable by LLL if some string zzz exists whereby exactly one of the strings xzxzxz and yzyzyz is a member of LLL;

otherwise, for every string zzz, we have xz∈Lxz ∈ Lxz∈L whenever yz∈Lyz ∈ Lyz∈L and we say that xxx and yyy are indistinguishable by LLL. If xxx and yyy are indistinguishable by LLL we write x≡Lyx ≡_L yx≡L​y. In other words, for x,y∈Σ∗x, y ∈ Σ^∗x,y∈Σ∗, x≡Lyx ≡_L yx≡L​y iff for each z∈Σ∗z ∈ Σ^∗z∈Σ∗ : xz∈L⇔yz∈Lxz ∈ L ⇔ yz ∈ Lxz∈L⇔yz∈L .

≡L≡_L≡L​ is an equivlance relation because obviously

  1. Reflexive: x≡Lxx ≡_L xx≡L​x;
  2. Symmetric: x≡Ly⇒y≡Lxx ≡_L y ⇒ y ≡_L xx≡L​y⇒y≡L​x;
  3. Transitive: x≡Ly∧y≡Lz⇒x≡Lzx ≡_L y ∧ y ≡_L z ⇒ x ≡_L zx≡L​y∧y≡L​z⇒x≡L​z.
Theorem

Let LLL be a language and let XXX be a set of strings. Say that XXX is pairwise distinguishable by LLL if every two distinct strings in XXX are distinguishable by LLL. Define the index of L to be the maximum number of elements in any set that is pairwise distinguishable by LLL. The index of L may be finite or infinite.

  1. If LLL is recognized by a DFA with kkk states, LLL has index at most kkk.
  2. If the index of LLL is a finite number kkk, it is recognized by a DFA with kkk states.
  3. LLL is regular iff it has finite index. Moreover, its index is the size of the smallest DFA recognizing it.

注意:

  • Myhill-Nerode Theorem 是 regular language 的充分必要條件
  • 可以找出某些 Pumping Lemma 無法辨別出來的 nonregular language

有了 Myhill-Nerode Theorem，我們可以證明一個 Languages 是 regular language 或是 nonregular languages:

  1. Given a langauge BBB, construct the equivalence relation ≡B≡_B≡B​ by using BBB.
  2. Prove that BBB has finite or infinite index:
  • Finite: BBB is a regular language.
  • Infinite: BBB is a nonregular language.

Minimization of DFAs

根據 Myhill-Nerode Theorem 我們可以知道 index of L 是能夠 accept LLL 的最小 DFA 的 states 數量，如果有一個 DFA 的 states 數量比 index of L 多，但是也能夠 accept LLL，這時這個 DFA 內部有一些 states 實際上是等價的，可以將其進行合併操作，使其最小化

Equivalent states: State ppp and state qqq are said to be
equivalent if for all w∈Σ∗w ∈ Σ^∗w∈Σ∗, δ^(p,w)∈F⇔δ^(q,w)∈F\hat δ(p, w) ∈ F ⇔ \hat δ(q, w) ∈ Fδ^(p,w)∈F⇔δ^(q,w)∈F ; and distinguishable otherwise.

Table-filling Algorithm

Given DFA A=(Q,Σ,δ,q0,F)A = (Q, Σ, δ, q_0, F )A=(Q,Σ,δ,q0​,F), the following algorithm finds all distinguishable pairs in AAA.

  • Basis: If p∈Fp ∈ Fp∈F and q∉Fq ∉ Fq∈/​F , {p,q}\{ p, q \}{p,q} is distinguishable.
  • Induction step: If {r,s}\{ r, s \}{r,s} is distinguishable, and r=δ(p,a)r = δ(p, a)r=δ(p,a), s=δ(q,a)s = δ(q, a)s=δ(q,a), for some a∈Σa ∈ Σa∈Σ, then {p,q}\{ p, q \}{p,q} is distinguishable.

實際操作: 以一個 table (類似 queue 的功能) 來紀錄要檢查的 pair

  • 先將 accept states 與其餘 states 組成的 pairs 加上標示 (數字)
  • 從 table 中取出 pair，往前推前一步的 distinguishable pairs，並將其標上數字
  • 重複直到 table 上有數字的 pair 都檢查過
  • table 上沒有被標記過的 pair 即為 Equivalent states pairs

得到 Equivalent state pairs: {A,E}\{ A, E \}{A,E}, {B,H}\{ B, H \}{B,H}, and {D,F}\{ D, F \}{D,F}.

Equivalent states in a given DFA can be found by the table-filling algorithm, and therefore, the DFA can be minimized by combining each set of equivalent states into one single state.

注意:

  • 如果要進行 Minimization 則找到的每個 Equivalent states pairs 都要進行合併，不能只合併某一些 pairs
