Formal Definition of CFGs

Definition

A context-free grammar is a 4-tuple $(V, ОЈ, R, S)$, where

  1. $V$ is a finite set called the variables;
  2. $ОЈ$ is a finite set, disjoint from $V$ ,called the terminals;
  3. $R$ is a finite set of rules (or productions), with each rule being a variable and a string of variables and terminals;
  4. $S вҲҲ V$ is the start variable.

Each rule consists of

  • head: variable in $V$
  • symbol вҶ’
  • body: a string in $(V вҲӘ ОЈ)^вҲ—$
  • If $u, v, w вҲҲ (V вҲӘ ОЈ)^вҲ—$, and $A вҶ’ w$ is a rule of the grammar, we say that $uAv$ yields $uwv$, written $uAv вҮ’ uwv$.
  • Say that $u$ derives $v$, written $u вҮ’^* v$, if $u = v$ or if a sequence $u_1, u_2, В·В·В· , u_k$ exists for $k вүҘ 0$ and $u вҮ’ u_1 вҮ’ u_2 вҮ’ В·В·В· вҮ’ v$ .
  • The language of the grammar is ${w вҲҲ ОЈ^вҲ—~|~S вҮ’^* w}$.

Ambiguity

Leftmost derivation: A derivation of a string w in a grammar G is a leftmost derivation if at every step the leftmost remaining variable is the one replaced.

Definition

A string $w$ is derived ambiguously in context-free grammar $G$ if it has two or more different leftmost derivations. Grammar $G$ is ambiguous if it generates some string ambiguously.

Inherently ambiguous: The context-free languages that can be generated only by ambiguous grammars. For example, ${a^ib^jc^k~|~i = j~or~j = k}$.

Chomsky Normal Form

Definition

A context-free grammar is in Chomsky normal form if every rule is of the form $A вҶ’ BC~or~A вҶ’ a$, where $a$ is any terminal and $A$, $B$, and $C$ are any variables except that $B$ and $C$ may not be the start variable. In addition we permit the rule $S вҶ’ Оө$, where $S$ is the start variable.

Theorem

Any context-free language is generated by a context-free grammar in Chomsky normal form.

  • иӯүжҳҺ: Convert any context-free grammar into Chomsky normal form:
    1. Add a new start variable;
    2. Eliminate all $Оө$ rules ($A вҶ’ Оө$);
    3. Handle all unit rules ($A вҶ’ B$);
    4. Convert all rules into the proper form.

  1. $Q$ is the set of states;

  3. $О“$ is the stack alphabet;
  4. $Оҙ$ : $Q Г— ОЈОө Г— О“Оө вҶ’ P(Q Г— О“_Оө)$ is the transition function;
  5. $q0 вҲҲ Q$ is the start state;
  6. $F вҠҶ Q$ is the set of accept states.

A pushdown automaton $M = (Q, ОЈ, О“, Оҙ, q0, F )$ accepts input $w$ if $w$ can be written as $w = w_1w_2 В·В·В· w_m$, where $w_i вҲҲ ОЈОө$ and sequences of states $r_0, r_1, В·В·В· , r_m вҲҲ Q$ and strings $s_0, s_1, В·В·В· , s_m вҲҲ О“^вҲ—$ exist that satisfy the following three conditions. Pumping Lemma for CFLs

Theorem

If $A$ is a context-free language, then there is a number p (the pumping length) where, if $s$ is any string in $A$ of length at least $p$, then $s$ may be divided into five pieces, $s = uvxyz$ satisfying the conditions:

  1. for each $i вүҘ 0$, $uv^ixy^iz вҲҲ A$;
  2. $|vy| > 0$; v, y дёҚеҗҢжҷӮзӮәз©әеӯ—дёІ
  3. $|vxy| вүӨ p$.

жңүдәҶ Pumping Lemma for CFLsпјҢжҲ‘еҖ‘еҸҜд»ҘеҲ©з”ЁеҸҚиӯүжі•еҺ»иӯүжҳҺдёҖеҖӢ Languages жҳҜ non-context-free languages:

  1. Assume the language, say, $B$, is context free in order to obtain a contradiction.
  2. By the pumping lemma for CFLs, a pumping length p exists, and any string $w вҲҲ B$ can be pumped if $|w| вүҘ p$.
  3. Find a string $s вҲҲ B$, $|s| вүҘ p$, that $s$ cannot be pumped as described in the pumping lemma. е°ӢжүҫеҸҚдҫӢ
  4. The contradiction is obtained, and therefore, $B$ is proved to be non-context-free.

OgdenвҖҷs Lemma for CFGs
