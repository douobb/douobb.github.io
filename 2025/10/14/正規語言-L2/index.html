Context-Free Grammars

CFGs: 上下文無關文法，指的是字元在進行替換時不需要考慮上下文，其擁有足夠的表現力來表示大多數程式設計語言的語法

Ambiguity

Leftmost derivation: A derivation of a string w in a grammar G is a leftmost derivation if at every step the leftmost remaining variable is the one replaced.

Definition

A string www is derived ambiguously in context-free grammar GGG if it has two or more different leftmost derivations. Grammar GGG is ambiguous if it generates some string ambiguously.

Inherently ambiguous: The context-free languages that can be generated only by ambiguous grammars. For example, {aibjck ∣ i=j or j=k}\{a^ib^jc^k~|~i = j~or~j = k\}{aibjck ∣ i=j or j=k}.

Chomsky Normal Form

Definition

A context-free grammar is in Chomsky normal form if every rule is of the form A→BC or A→aA → BC~or~A → aA→BC or A→a, where aaa is any terminal and AAA, BBB, and CCC are any variables except that BBB and CCC may not be the start variable. In addition we permit the rule S→εS → εS→ε, where SSS is the start variable.

如果原有的 grammar 包含 S→εS → εS→ε，則有兩種處理方式:

  1. L(G′)=L(G)−{ε}L(G') = L(G) - \{ε\}L(G′)=L(G)−{ε}
  2. 允許 S→εS → εS→ε rule
Theorem

Any context-free language is generated by a context-free grammar in Chomsky normal form.

  • 證明: Convert any context-free grammar into Chomsky normal form:
    1. Add a new start variable;
    2. Eliminate all εεε rules (A→εA → εA→ε);
    3. Handle all unit rules (A→BA → BA→B);
    4. Convert all rules into the proper form.

Equivalence with CFGs

Theorem

A language is context free if and only if some pushdown automaton recognizes it.

Lemma

If a language is context free, then some pushdown automaton recognizes it.

  • 證明: Convert the given CFG into a PDA.
    • 利用 PDA 的 nondeterminism 特性，用 leftmost derivation 產生所有可能的 substitution sequence， 將 intermediate strings 儲存在 stack 中並且與 input string 進行對應，如果能到達 Accept states 代表能夠接受
    • For the symbol on the stack top, two kinds of transitions:
      • For terminal aaa, match the input: a,a→εa, a → εa,a→ε
      • For variable AAA, apply the rule: ε,A→wε, A → wε,A→w

Pumping Lemma for CFLs

Theorem

If AAA is a context-free language, then there is a number p (the pumping length) where, if sss is any string in AAA of length at least ppp, then sss may be divided into five pieces, s=uvxyzs = uvxyzs=uvxyz satisfying the conditions:

  1. for each i≥0i ≥ 0i≥0, uvixyiz∈Auv^ixy^iz ∈ Auvixyiz∈A;
  2. ∣vy∣>0|vy| > 0∣vy∣>0; v, y 不同時為空字串
  3. ∣vxy∣≤p|vxy| ≤ p∣vxy∣≤p.
  • 證明: 類似 Pumping Lemma for RL 的證明方式，透過鴿籠原理，當 parse tree 的樹高足夠時 (> variables 總數)，必定存在 variable R 在一條路徑中經過兩次以上，因此可以對其產生出的 string 進行 Pumping 而仍然屬於 CFL

注意:

  • context-free language 一定滿足 Pumping Lemma for CFLs
  • Pumping Lemma for CFLs 是 context-free language 的必要條件，不是充分條件
    • 如果一個 language 滿足 Pumping Lemma for CFLs，其不一定為 context-free language

有了 Pumping Lemma for CFLs，我們可以利用反證法去證明一個 Languages 是 non-context-free languages:

  1. Assume the language, say, BBB, is context free in order to obtain a contradiction.
  2. By the pumping lemma for CFLs, a pumping length p exists, and any string w∈Bw ∈ Bw∈B can be pumped if ∣w∣≥p|w| ≥ p∣w∣≥p.
  3. Find a string s∈Bs ∈ Bs∈B, ∣s∣≥p|s| ≥ p∣s∣≥p, that sss cannot be pumped as described in the pumping lemma. 尋找反例
  4. The contradiction is obtained, and therefore, BBB is proved to be non-context-free.
