Decidable Problems Concerning Regular Languages

定義 ADFA={⟨B,w⟩ ∣ B is a DFA that accepts input string w}A_{DFA} = \{ ⟨B, w⟩~|~B~is~a~DFA~that~accepts~input~string~w \}ADFA​={⟨B,w⟩ ∣ B is a DFA that accepts input string w}

Theorem

ADFAA_{DFA}ADFA​ is a decidable language.

  • 證明: 建立一個 TM MMM 模擬 www 在 BBB 上的行為，如果停在 accept states 則 ADFAA_{DFA}ADFA​ appect，否則會 reject，只有這兩種可能，因此是 Decidable 的

定義 ANFA={⟨B,w⟩ ∣ B is a NFA that accepts input string w}A_{NFA} = \{ ⟨B, w⟩~|~B~is~a~NFA~that~accepts~input~string~w \}ANFA​={⟨B,w⟩ ∣ B is a NFA that accepts input string w}

Theorem

ANFAA_{NFA}ANFA​ is a decidable language.

  • 證明: 建立一個 TM NNN，並建立與 BBB 等價的 DFA CCC，可以將 ⟨C,w⟩⟨C, w⟩⟨C,w⟩ 帶入 TM MMM 去判斷是 accept or reject

定義 AREX={⟨R,w⟩ ∣ R is a regular expression that generates string w}A_{REX} = \{ ⟨R, w⟩~|~R~is~a~regular~expression~that~generates~string~w \}AREX​={⟨R,w⟩ ∣ R is a regular expression that generates string w}

Theorem

AREXA_{REX}AREX​ is a decidable language.

  • 證明: 建立一個 TM PPP，並建立與 RRR 等價的 NFA AAA，可以將 ⟨A,w⟩⟨A, w⟩⟨A,w⟩ 帶入 TM NNN 去判斷是 accept or reject

定義 EDFA={⟨A⟩ ∣ A is a DFA and L(A)=∅}E_{DFA} = \{ ⟨A⟩~|~A~is~a~DFA~and~L(A) = ∅ \}EDFA​={⟨A⟩ ∣ A is a DFA and L(A)=∅}

Theorem

EDFAE_{DFA}EDFA​ is a decidable language.

  • 證明: 建立一個 TM TTT，從 start state 開始進行 BFS 並 mark 已經被經過的 states，如果有到達 accept states 則 reject，否則 accept (沒有 string 會被 AAA accept)

定義 EQDFA={⟨A,B⟩ ∣ A and B are DFAs and L(A)=L(B)}EQ_{DFA} = \{ ⟨A, B⟩~|~A~and~B~are~DFAs~and~L(A) = L(B) \}EQDFA​={⟨A,B⟩ ∣ A and B are DFAs and L(A)=L(B)}

Theorem

EQDFAEQ_{DFA}EQDFA​ is a decidable language.

  • 證明: 建立一個 TM FFF，根據某些操作在 DFA 上的封閉性，可以建立一個 DFA CCC 使得 L(C)=(L(A)∩L(B)‾)∪(L(A)‾∩L(B))L(C) = (L(A)∩\overline {L(B)})∪(\overline {L(A)}∩L(B))L(C)=(L(A)∩L(B)​)∪(L(A)​∩L(B))，代表是兩個 language 的對稱差，可以將 ⟨C⟩⟨C⟩⟨C⟩ 帶入 TM TTT 去判斷其 language 是否為空，如果 TM TTT 是 accept 則 accept，否則 reject (對稱差不為空)

Decidable Problems Concerning Context-Free Languages

定義 ACFG={⟨G,w⟩ ∣ B is a CFG that generates string w}A_{CFG} = \{ ⟨G, w⟩~|~B~is~a~CFG~that~generates~string~w \}ACFG​={⟨G,w⟩ ∣ B is a CFG that generates string w}

Theorem

ACFGA_{CFG}ACFG​ is a decidable language.

  • 證明: 建立一個 TM SSS，先將 GGG 轉換為等價的 Chomsky normal form，如果 www 的長度為 n (不為 0)，則在 2n - 1 步內一定可以判斷是否能產生 www，如果能產生則 accept，否則 reject
    • 也能輕易地用 CYK Algorithm 來判斷

The Diagonalization Method

Measuring the sizes of infinite sets.

假設有兩個 set A,BA, BA,B，如果能找到一個函數 fff from AAA to BBB，並滿足:

  • one-to-one: $ x ≠ y ⇒ f (x) ≠ f (y)$
  • onto: $ ∀b ∈ B, ∃a ∈ A, f (a) = b$

則 A,BA, BA,B 為 same size，並且 fff 稱為 correspondence

Definition

A set AAA is countable if either it is finite or it has the same size as NNN .

Theorem

RRR is uncountable.

  • 證明: 先假設實數集合 RRR 是 countable 的，則存在函數 fff 使得 NNN 與其內元素一一對應，我們可以找到一個實數 xxx 是沒有被對應的的，產生矛盾
    • xxx 的產生方式: 取 i 對應元素的小數點後第 i 位，並改變其數值
Corollary

Some languages are not Turing-recognizable.

  • 證明: 利用 Diagonalization Method

A Turing-Unrecognizable Language

A language is co-Turing-recognizable if it is the complement of a Turing-recognizable language.

Theorem

A language is decidable iff it is Turing recognizable and co-Turing-recognizable.

  • 證明: 建立一個 TM MMM，當接收 string www 後，同時模擬 M1,M2M_1, M_2M1​,M2​ 的行為，如果 M1M_1M1​ accept 則 accept，如果 M2M_2M2​ accept 則 reject (可以避免卡在 loop)
    • M1M_1M1​ 是 Turing recognizable 所對應的 TM
    • M2M_2M2​ 是 co-Turing-recognizable 所對應的 TM
    • 不可能兩者同時卡在 loop
Corollary

ATM‾\overline{A_{TM}}ATM​​ is not Turing-recognizable.

  • 證明: 已知 ATMA_{TM}ATM​ is undecidable 且 ATMA_{TM}ATM​ is Turing-recognizable
