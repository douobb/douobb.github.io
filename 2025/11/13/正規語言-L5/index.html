L5 Reducibility

If problem A reduces to problem B, we can use a solution to B to solve A.

When A is reducible to B, solving A cannot be harder than solving B because a solution to B gives a solution to A. Therefore, if A is reducible to B,

  • B is decidable ⇒ A is decidable;
  • A is undecidable ⇒ B is undecidable.

Reductions via Computation Histories

The computation history for a Turing machine on an input is simply the sequence of configurations that the machine goes through as it processes the input.

Definition

An accepting computation history for MMM on www is a sequence of configurations, C1,C2,...,ClC_1, C_2, ... , C_lC1​,C2​,...,Cl​, where C1C_1C1​ is the start configuration of MMM on www, ClC_lCl​ is an accepting configuration of MMM , and each CiC_iCi​ legally follows from Ci−1C_{i−1}Ci−1​ according to the rules of MMM .

A rejecting computation history for MMM on www is defined similarly, except that ClC_lCl​ is a rejecting configuration.

Definition

A linear bounded automaton is a restricted type of Turing machine wherein the tape head isn’t permitted to move off the portion of the tape containing the input. (除了 input 之外不需要額外空間)

Lemma

Let MMM be an LBA with qqq states and ggg symbols in the tape alphabet. There are exactly qngnqng^nqngn distinct configurations of MMM for a tape of length nnn.

Mapping Reducibility

Being able to reduce problem AAA to problem BBB by using a mapping reducibility means that a computable function, called a reduction, exists that converts instances of problem AAA to instances of problem BBB.

Computable Functions

A Turing machine computes a function by starting with the input to the function to the tape and halting with the output of the function on the tape.

Definition

A function f:Σ∗→Σ∗f: Σ^∗ → Σ^∗f:Σ∗→Σ∗ is a computable function if some Turing machine MMM, on every input www, halts with just f(w)f(w)f(w) on its tape.

Formal Definition of Mapping Reducibility

Definition

Language AAA is mapping reducible to language BBB, written A≤mBA ≤_m BA≤m​B, if there is a computable function f:Σ∗→Σ∗f: Σ^∗ → Σ^∗f:Σ∗→Σ∗, where for every www, w∈A⇔f(w)∈Bw ∈ A ⇔ f(w) ∈ Bw∈A⇔f(w)∈B .
The function fff is called the reduction of AAA to BBB.

Theorem

If A≤mBA ≤_m BA≤m​B and BBB is decidable, then AAA is decidable.

  • 證明: 假設 BBB 的 decider MMM 存在，則可以利用 MMM 以及 computable function fff 建立 AAA 的 decider NNN
Corollary

If A≤mBA ≤_m BA≤m​B and AAA is undecidable, then BBB is undecidable.

Theorem

If A≤mBA ≤_m BA≤m​B and BBB is Turing-recognizable, then AAA is Turing-recognizable.

  • 證明: 類似 decidable 的證明
Corollary

If A≤mBA ≤_m BA≤m​B and AAA is not Turing-recognizable, then BBB is not Turing-recognizable.
