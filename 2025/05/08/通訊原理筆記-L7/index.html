L7 Multiple Division Techniques

Motivation: to have many traffic channels

分類:

  • Frequency: FDMA (Frequency Division Multiple Access)
  • Time: TDMA (Time Division Multiple Access)
  • Code: CDMA (Code Division Multiple Access)
  • SDMA (Space Division Multiple Access)

Models of Multiple Divisions

  • Forward (Downlink) Channel: 基地台 (BS) -> 行動裝置 (MS)
  • Reverse (Uplink) Channel: MS -> BS
  • Duplex Communications (雙向通訊)
    • Full Duplex (全雙工) - 上下行可以同時進行
    • Half Duplex (半雙工) - 上下行不能同時進行，只能交替發送與接收
    • Type:
      • FDD (frequency division duplexing) - 上下行使用不同頻率
      • TDD (time division duplexing) - 上下行使用相同頻率，但在不同的時間傳送

FDMA

每個使用者被分配一個獨立的頻率通道，且通訊可同時進行

  • Structure:
  • 頻道之間用 Guard Bands 分隔，避免干擾
  • 通常搭配 FDD

TDMA

每個使用者交替使用相同頻率，在不同時間點進行通訊

  • Structure:
  • 傳輸時間之間用 Guard Time 分隔，避免干擾
  • TDMA/FDD:
  • TDMA/TDD:

CDMA

每個使用者同時使用同一段頻率，使用 spectrum-spread technology

  • 每個使用者有獨特的 spread-spectrum code (擴頻碼)
  • 資料在傳送前先乘上擴頻碼，接收端使用同樣的擴頻碼進行解碼
  • Sequence generation - 如何產生擴頻碼:
  • Near far problem
    • 不同 MS 到 BS 的距離不同，導致 SNR 不同，可能把距離較遠的 MS 誤認為雜訊
    • 解決方式: Power Control - 距離近的 MS 用較小的 power 傳輸

Spread Spectrum Modulation

將原始訊號的頻率展開到比原本寬很多的頻寬，提高抗干擾能力與保密性

  • DSSS (Direct-Sequence spread spectrum)
    • 訊號乘上一個 Pseudo Noise (PN)
    • PN: 擁有很多持續時間非常短的脈衝 (pulses)，擁有較大的頻寬
    • 解碼時同樣乘上 PN，之後用 low-pass filter 把高頻的雜訊濾掉
  • FHSS (Frequency Hopping Spread Spectrum)
    • 傳送訊息時，不斷在不同頻率的載波上切換
    • Type:
      • Fast hopping: multiple hops per data bit
      • Slow hopping: multiple data bits per hop
    • 可能發生碰撞 (不同使用者跳到相同頻率)

SDMA

利用空間來區分不同使用者的訊號，使用者可以同時使用同一段頻率

  • 使用 smart antennas - 能動態調整訊號方向與形狀的天線系統，由多個天線組成

OFDM & OFDMA

OFDM (Orthogonal Frequency Division Multiplexing): 將數據分成多個正交的子載波並行傳輸

  • 每個子載波之間不會互相干擾 (正交)
  • 子載波有較低的 symbol rate (符號速率) - 可以有效應對多重路徑效應造成的 ISI

OFDMA: 基於 OFDM 技術的多重存取方法

  • 將子載波分配給不同的使用者
  • 使用者可以在較低的速率下進行並行通信
  • Difference between FDM & OFDM
